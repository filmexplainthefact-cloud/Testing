FAQ
How to deposit money?
Go to Wallet → Add Money. Scan the QR code or use UPI ID to pay. Enter UTR number and optionally upload payment screenshot. Admin will approve within a few hours.
Room ID kab milega?
Room ID and Password tournament start hone se pehle My Matches section mein dikhega. Notification bhi aayegi. App refresh karte rehna.
Withdrawal kitne time mein hoga?
Withdrawal request submit karne ke 24 ghante ke andar process ho jaata hai. Minimum withdrawal amount ₹100 hai.
Referral bonus kab milega?
Jab aapka referred friend account banata hai aur login karta hai, turant ₹10 dono ke accounts mein credit ho jaata hai.
Tournament mein kaise join karein?
Home page pe tournament card dekho, "Join Now" dabao. Wallet balance aur fee confirm karke join karo. Joined tournament My Matches mein dikhega.
